# 🤖 EzBot_IA

Bot de Minecraft com Mineflayer e IA que aprende a jogar sozinho!

## Funcionalidades
- 9 IAs em rotação automática
- Auto-come, auto-armadura, auto-espada
- Sabe HP de todos os mobs
- Tenta zerar o jogo e matar o Ender Dragon

## Como usar
Clone o repositório, adicione suas keys no .env e rode!
